#Student Course Enrolment & Timetable Management System

This project is a simple Java application that allows students to enroll in courses and view their timetable. The system uses Oracle Database for storing student, course, and timetable records. The project is developed in IntelliJ IDEA with a GUI. 

## Features 
Add and manage students
Add and manage courses 
Student course enrolment
View timetable
Update and delete records
Oracle database connectivity 

## Technologies Used
Java 
IntelliJ IDEA 
Swing/JavaFX GUI 
Oracle Database 
JDBC 

## How to Run 
1. Open the project in IntelliJ IDEA.
2. Set your Oracle Database username and password in the DB connection file.
3. Run the SQL scripts to create tables.
4. Run the Main class to start the application.

## Database Setup 
Create tables using the provided SQL scripts. 
Connect to Oracle using JDBC.
Make sure the Oracle service is running before starting the program
